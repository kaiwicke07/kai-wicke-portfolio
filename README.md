# Kai Wicke - Portfolio

Personal portfolio website built with React, TypeScript, and Tailwind CSS.

## Development

```bash
pnpm install
pnpm run dev
```

## Deploy

Push to `main` branch — GitHub Actions will automatically build and deploy to GitHub Pages.

## Tech Stack

- React 19
- TypeScript
- Tailwind CSS v4
- Vite
- Wouter (routing)
